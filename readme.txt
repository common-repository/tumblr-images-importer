== Description ==

This plugin will allow to import images from any Tumblr blog into wordpress.
It will automatically create the posts with the image set as Featured Image.

You can also set the post format (Standard, gallery, etc), the ammount of images to be inserted and the status of the post (Published, Draft).


== Installation ==

1. Upload `images-tumblr-importer.zip` contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! You can now access the plugin options through Settings menu.
